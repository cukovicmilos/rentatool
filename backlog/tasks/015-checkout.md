# Checkout proces

- **Status**: To Do
- **Labels**: frontend, backend
- **Milestone**: M4-Korpa-Checkout

## Opis

Implementirati checkout sa podacima musterije i opcijama dostave.

## Zadaci

- [ ] Forma sa podacima musterije:
  - Ime i prezime (obavezno)
  - Email (obavezno, validacija formata)
  - Telefon (obavezno)
  - Adresa (obavezno ako je dostava)
- [ ] Izbor dostave:
  - Licno preuzimanje (besplatno)
  - Dostava (10 EUR)
  - Dostava + preuzimanje (15 EUR)
- [ ] Pregled narudzbe pre potvrde
- [ ] Ukupna cena sa svim troskovima
- [ ] CSRF zastita
- [ ] Validacija dostupnosti alata pre finalizacije
